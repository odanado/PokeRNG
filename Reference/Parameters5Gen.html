Parameters5Gen

namespace PokeRNG {
    template<typename Constant>
    class Parameters5Gen;
}

概要

5世代乱数の初期シード決定に使われるパラメータ群を保持するテンプレートクラスである。
このクラスのインスタンスをCalc5GenSeedに渡すことで初期シードの計算を行える。
